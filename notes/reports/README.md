# 한국투자증권 리포트 분석 노트

## 목적

이 폴더는 **한국투자증권에서 발행하는 일일 리포트를 읽고 분석한 수기 기록**을 보관합니다.

### 핵심 목표
- 매 영업일 기준 **3개의 리포트**를 읽고 요약
- 시장 동향에 대한 **개인적 분석과 해석** 기록
- 투자 판단에 대한 **피드백과 성찰** 축적

## 파일 구조

```
notes/reports/
├── README.md              # 본 문서
└── YY-MM-DD~DD_기록.pdf   # 날짜 범위별 수기 기록
```

### 파일 명명 규칙
- 형식: `YY-MM-DD~DD_기록.pdf`
- 예시: `26-01-08~09_기록.pdf` (2026년 1월 8일~9일 기록)

## 기록 구성

각 일자별 기록은 다음 섹션으로 구성됩니다:

### 1. 리포트 요약 (3개/일)
| 카테고리 | 내용 |
|---------|------|
| 경제분석 Note | 거시경제 동향, 환율, 금리 등 |
| 채권분석 Note | 채권시장, 금통위, 수급 분석 |
| 차이나 Note | 중국 경제 및 시장 동향 |
| 자산배분전략 | 포트폴리오 관점의 전략 |
| 글로벌 Brief | 해외 시장 주요 이슈 |

### 2. After Market
- 장 마감 후 주요 뉴스 및 시장 반응
- 지정학적 이슈, 섹터별 강세/약세 분석

### 3. 개인 피드백
- 리포트 내용에 대한 개인적 해석
- 투자 판단과 그에 대한 성찰
- 시장 심리 분석 (희망 → 낙관 → 흥분 → 열광 → 도취 사이클 등)

## 활용 방법

1. **시장 학습**: 전문가 리포트를 통한 시장 분석 역량 강화
2. **기록 축적**: 시간이 지난 후 예측의 정확도 검증
3. **투자 성찰**: 감정적 판단과 합리적 분석 구분 훈련
4. **패턴 인식**: 반복되는 시장 상황에 대한 대응력 향상

## OCR 호환성

- 수기 작성이지만 정돈된 필체로 **OCR 인식률 85-90%** 수준
- 날짜, 섹션 구분, 핵심 키워드 등 구조화되어 있어 자동 요약 생성 가능
