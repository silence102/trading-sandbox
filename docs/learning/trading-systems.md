# 트레이딩 시스템 개념 및 증권사별 비교

> HTS, MTS, WTS 개념 정리와 주요 증권사 서비스 비교 분석

## 1. 트레이딩 시스템 개념

### HTS (Home Trading System) - 홈 트레이딩 시스템

집이나 사무실에서 주식과 파생상품 등 금융 투자를 할 수 있게 해주는 **PC용 프로그램**입니다.

| 항목 | 내용 |
|------|------|
| 정의 | PC에 설치하는 종합 주식거래 프로그램 |
| 장점 | 다양한 정보, 강력한 차트 기능, 빠른 매매 |
| 단점 | 주로 Windows 전용, 설치 필요 |
| 점유율 | 유가증권시장 50% 이상, 코스닥 90% 이상 |

### MTS (Mobile Trading System) - 모바일 트레이딩 시스템

스마트폰이나 태블릿에서 주식 거래를 할 수 있게 해주는 **모바일 앱**입니다.

| 항목 | 내용 |
|------|------|
| 정의 | 스마트폰/태블릿용 주식거래 앱 |
| 장점 | 이동 중 거래 가능, 간편한 접근 |
| 단점 | 화면 제한, 정보량 제한 |
| 점유율 | 전체 거래수단 중 **50% 이상** (2020년 이후) |

### WTS (Web Trading System) - 웹 트레이딩 시스템

별도 설치 없이 **웹 브라우저**를 통해 주식 거래를 할 수 있는 시스템입니다.

| 항목 | 내용 |
|------|------|
| 정의 | 웹 브라우저 기반 주식거래 시스템 |
| 장점 | 설치 불필요, OS 무관 (Mac 지원), 큰 화면 |
| 단점 | HTS보다 기능 제한적 |
| 트렌드 | MZ세대 Mac 사용자 증가로 수요 확대 |

### 비교 요약표

| 구분 | HTS | MTS | WTS |
|------|-----|-----|-----|
| **사용 기기** | PC (Windows) | 스마트폰/태블릿 | 웹 브라우저 |
| **설치** | 프로그램 설치 | 앱 설치 | **설치 불필요** |
| **정보량** | **가장 많음** | 제한적 | 중간 |
| **OS 호환성** | Windows 전용 | iOS/Android | **범용 (Mac 포함)** |
| **화면 크기** | 큼 | 작음 | 큼 |
| **이동성** | 낮음 | **높음** | 중간 |

## 3. 한국투자증권 eFriend Plus 상세

> 현재 사용 중인 HTS

### 주요 기능

#### 1. 종목검색
- 재무분석지표, 기술적 지표 조합 검색
- **실시간 검색** 기능 제공
- 데이트레이더, 스캘퍼 등 다양한 투자 성향 지원

#### 2. 종합주문
- 시세포착 → 시세확인 → 주문 → 주문확인 일괄 처리
- 통합 주문 설정 기능
- 주문 착오 방지 설정

#### 3. 멀티분석형 차트
- 여러 종목 동시 모니터링
- 분할 차트 간 동기화 기능
- 한 종목 다각도 분석

#### 4. 관심종목 관리
- 그룹당 최대 **100개** 종목 등록
- 실시간 시세 처리 최적화
- 강력한 편집 기능

#### 5. 추가 기능
- 연속조회, 메뉴검색, 일정관리
- 사용자 저장화면 (나만의 매매스타일)
- 국내외 파생상품 및 해외주식 통합 거래

### 시스템 요구사항
- **Windows 10 64Bit 이상**

## 4. 키움증권 영웅문 상세

> 국내 HTS 시장점유율 **1위** (약 28%)

### 주요 특징

#### 1. 단축코드 시스템
| 코드 | 기능 |
|------|------|
| `4989` | 매수/매도 창 |
| `8282` | 호가 주문창 |

- 네 자리 숫자로 빠른 화면 접근
- 숙련자에게 유리한 자유도

#### 2. 조건검색
- HTS에서 작성한 조건식 사용
- **실시간 조건검색** 지원 (MTS 포함)
- 최대 10개 조건식, 100종목 제한

#### 3. 자동 감시 주문
- 감시조건 최대 **40개** 설정
- 유효기간 동안 자동 주문 실행
- 직장인/대학생 투자자에게 유용

#### 4. HTS-MTS 연동
- 관심종목 그룹 동기화
- 두 시스템 병행 사용 시 편리

### MTS 영웅문S# (2022년 대개편)
- 국내주식 + 해외주식 **통합 앱**
- UI/UX 대폭 개선
- 가독성 향상, 반응속도 개선

### HTS 최신 기능
- 실시간 마켓(1597) 화면
- 60개 이상 미니 전환화면
- 쾌속주문 툴바

## 5. 기타 증권사 서비스 비교

### MTS 앱 비교

| 증권사 | 앱 이름 | MAU | 특징 |
|--------|---------|-----|------|
| 키움증권 | 영웅문S# | - | 기능 특화, HTS 연동 |
| 삼성증권 | mPOP | 234만 | 깔끔한 UI, 다양한 차트 |
| 미래에셋증권 | M-STOCK | 237만 | 빠른 반응속도, 미려한 디자인 |
| 한국투자증권 | eFriend Smart | - | HTS 연동, 해외주식 통합 |
| 토스증권 | 토스 | - | 간단하고 직관적, 초보자 친화 |

### 토스증권 특징
- **설치 불필요** WTS 제공
- 간단하고 직관적인 인터페이스
- AI 기반 어닝콜 실시간 번역
- 소수점 주식 거래 (최소 1,000원)
- 해외주식 수수료 수익 **업계 1위** (2025년 2분기)

### 삼성증권 WTS
- POP HTS Web
- Mac 사용자 지원
- 설치 없이 웹 브라우저에서 거래

## 7. 한국투자증권 vs 키움증권 요약

| 구분 | 한국투자증권 | 키움증권 |
|------|-------------|---------|
| **HTS** | eFriend Plus | 영웅문 |
| **MTS** | eFriend Smart | 영웅문S# |
| **WTS** | 미제공 | 미제공 |
| **API** | KIS Developers (REST) | Open API+ (OCX + REST) |
| **조건검색** | ❌ | ✅ |
| **해외파생** | ✅ | ❌ |
| **시장점유율** | 상위권 | **1위 (28%)** |

### 결론

**현재 사용 중인 한국투자증권 eFriend Plus**는:
- 종목검색, 멀티분석 차트 등 강력한 기능 제공
- 해외주식 통합 거래 가능
- **REST API (KIS Developers)** 와 연계하여 자동매매 구현에 적합

**키움증권 영웅문**이 필요한 경우:
- HTS 조건검색 기능이 필수일 때
- 단축코드 기반 빠른 매매가 필요할 때
- 기존 영웅문 사용 경험이 있을 때
