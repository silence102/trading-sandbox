# AI 요약 프롬프트 가이드

> 이 파일을 수정하여 Claude API 요약 방식을 커스터마이즈할 수 있습니다.

---

## 현재 프롬프트

```
당신은 투자 분석 전문가입니다. 아래 한국투자증권 리서치 리포트들을 분석하여 종합 일일 리포트를 작성해주세요.

## 분석 요청사항

1. **핵심 요약**: 오늘의 주요 이슈 3-5개를 간결하게 정리
2. **테마별 분석**: 각 리포트의 주요 내용을 테마(경제, 채권, 글로벌, 중국 등)로 분류
3. **연관관계 분석**: 여러 리포트에서 공통적으로 언급되는 이슈나 상호 연관된 내용 파악
4. **투자 인사이트**: 실질적인 투자 판단에 도움이 되는 핵심 포인트
5. **주의사항**: 리스크 요인이나 불확실성 요소

## 출력 형식

마크다운 형식으로 작성해주세요:
- 헤더(#, ##, ###)를 활용한 구조화
- 불릿 포인트로 핵심 내용 정리
- 중요한 수치나 날짜는 강조(**굵게**)
- 투자 인사이트는 > 인용 블록 사용
```

---

## 커스터마이즈 가이드

### 분석 관점 추가/수정

원하는 분석 관점을 추가하거나 수정할 수 있습니다:

```
## 분석 요청사항

1. **핵심 요약**: ...
2. **섹터별 영향**: IT, 반도체, 바이오 등 섹터별 영향 분석
3. **매매 타이밍**: 단기/중기 관점의 매매 시점 제안
4. **포트폴리오 조정**: 현재 보유 종목에 대한 영향
```

### 출력 형식 변경

출력 형식을 변경할 수 있습니다:

```
## 출력 형식

1. 요약은 500자 이내로
2. 각 섹션별로 명확한 구분선 사용
3. 핵심 수치는 테이블로 정리
4. 결론은 한 문장으로
```

### 특정 관심사 반영

개인 투자 스타일에 맞게 조정:

```
## 추가 요청사항

- 배당주 관점에서 분석
- ISA 계좌 투자자 관점 반영
- 중장기 가치투자 관점 강조
- 환율 영향 상세 분석
```

---

## 프롬프트 적용 방법

1. 이 파일(`summary_prompt_guide.md`)에서 프롬프트 수정
2. `scripts/report_summarizer.py`의 `SUMMARY_PROMPT` 변수에 반영
3. 또는 향후 이 파일을 자동으로 읽어오도록 개선 예정

---

## 예시 출력 포맷

```markdown
# 일일 리서치 리포트 종합

**날짜**: 2026-02-02
**분석 리포트 수**: 3개

---

## 핵심 요약

1. **미국 고용 지표 호조** - 연준 금리 동결 기대 강화
2. **중국 경기 부양책 기대** - 춘절 이후 정책 발표 예상
3. **국내 반도체 수출 호조** - SK하이닉스 목표가 상향

---

## 테마별 분석

### 경제/거시
- 미국 1월 고용지표 예상치 상회
- 달러-원 환율 1,430원대 안정

### 채권/금리
- 국고채 3년물 금리 3.2% 수준 유지
- 금통위 동결 전망 우세

---

## 투자 인사이트

> 반도체 섹터 비중 확대 고려.
> 다만 미국 관세 리스크 모니터링 필요.

---

## 주의사항

- 트럼프 관세 정책 불확실성
- 중국 부동산 리스크 지속
```
