# 증권사 API 커스터마이징 계획

## 개요

KIS Developers API를 활용한 커스터마이징 및 자동매매 시스템 구현 계획

## 현재 상태

- [x] KIS Developers 실전투자 API Key 발급
- [x] KIS Developers 모의투자 API Key 발급
- [ ] API 연동 테스트
- [ ] 커스터마이징 기능 구현

## 구현 예정 기능

### 1. 기본 기능
- [ ] 계좌 조회
- [ ] 잔고 조회
- [ ] 주문 (매수/매도)
- [ ] 주문 조회/취소

### 2. 자동화 기능
- [ ] 조건부 주문
- [ ] 분할 매수/매도
- [ ] 손절/익절 자동화

### 3. 분석 기능
- [ ] 포트폴리오 분석
- [ ] 수익률 계산
- [ ] 거래 내역 분석
