# 일일 마켓 브리핑 자동화 확장 계획

## 개요

현재 구현된 일일 브리핑 자동화를 확장하여 더 다양한 데이터 소스를 포함할 계획

## 현재 구현 상태

| 기능 | 데이터 소스 | 구현 | API 키 |
|------|------------|------|--------|
| 뉴스 수집 | RSS (매일경제, 한경, 이데일리) | ✅ 완료 | 불필요 |
| 증시 정보 | PyKRX (KOSPI, KOSDAQ, 관심종목) | ✅ 완료 | 불필요 |
| 기업 공시 | DART API | ✅ 완료 | ✅ 발급 완료 |
| 경제 지표 | ECOS API | ✅ 완료 | ✅ 발급 완료 |
| AI 분석 | ChatGPT OpenAI API (변경 예정) | ✅ 완료 | 📝 연동 예정 |

> **참고**: AI 분석은 기존 Anthropic Claude API에서 **ChatGPT OpenAI API로 변경 예정**

## 확장 예정 기능

### 1. 데이터 소스 추가
- [ ] 해외 증시 정보 (미국, 중국, 유럽)
- [ ] 환율/금리 실시간 정보
- [ ] 원자재 가격 (금, 유가 등)
- [ ] 암호화폐 시세 (선택적)

### 2. 분석 기능 강화
- [ ] 종목별 기술적 분석 지표 추가
- [ ] 섹터별 자금 흐름 분석
- [ ] 외국인/기관 수급 분석

### 3. 리포트 자동화
- [ ] 증권사 리포트 요약 (가능한 범위 내)
- [ ] 주요 이슈 키워드 추출
- [ ] 일일/주간 트렌드 분석

### 4. 알림 시스템
- [ ] 급등/급락 종목 알림
- [ ] 관심종목 목표가 도달 알림
- [ ] 중요 공시 실시간 알림

## 기술적 고려사항

### 데이터 소스 우선순위

| 우선순위 | 소스 | 이유 |
|---------|------|------|
| 높음 | 해외 증시 | 국내 시장 선행 지표 |
| 높음 | 환율/금리 | 거시 경제 영향 |
| 중간 | 섹터별 분석 | 투자 판단 보조 |
| 낮음 | 암호화폐 | 선택적 관심사 |

### API 키 발급 현황

| API | 상태 | 비고 |
|-----|------|------|
| DART | ✅ 발급 완료 | .env에 등록 완료 |
| ECOS | ✅ 발급 완료 (비영리) | .env에 등록 완료 |
| OpenAI (ChatGPT) | 📝 연동 예정 | Anthropic → OpenAI 변경 |

### API 제한 사항
- PyKRX: 저작권 주의 (상업적 사용 시 KRX OPEN API 검토)
- DART: 일일 호출 제한 존재
- ECOS: 안정적, 제한 적음
